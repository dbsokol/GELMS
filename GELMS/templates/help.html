What is GELMS?

GELMS, a Generic Electronic Learning Management System is a Client-Server based web application that needs access to the internet connection and it needs a web browser to open the application, Google Chrome, Mozilla Firefox or Safari anything would work. Also, this application is made keeping in mind the busy schedule of person, so this even works on a mobile device running on any platform like Android, iOS etc. Also, a person should be enrolled in the respective institution using our platform to access the features of our application.

Product Features

Admin Features:

  • Admins can add and remove courses
  • Admins can add and remove students from specific course
  • Admins can assign a teacher and/or grader to a specific course

Teacher Features:

  • Teachers can post, edit, and remove a syllabus to a specific course
  • Teachers can view a list of all people (students, teachers, graders) in a specific course
  • Teachers can turn reader mode on and off
  • Teachers can post, edit, and remove announcements to specific course

Student Features:

  • Students can view announcements for a specific course
  • Studetns can view a syllabus for a specific course
  • Studetns can turn reader mode on or off

Usage

Landing Page

  • This is the landing page
  • Regular users can log in to the application from here
  • Admins must visit the following URL to log in: http://gelms.pythonanywhere.com/admin

Admin Log In:

  • This is the admin login page
  • The default admin username is superuser and the default password is superuser

Admin Home Page

  • This is the initial admin interface where the admin can add/remove users, courses and tools

Admin User Management

  • This is the add user interface
  • The button on the top right allows the admin to create a new user
  • Clicking on a user email allows the admin to edit an existing user

Admin Course Management

  • This is the add course interface
  • The button on the top right allows the admin to create a new course
  • Clicking on a course name allows the admin to edit an existing course

Admin Tool Management

  • This is the add tools interface
  • The add button on the right allows the admin to add a new tool
  • Clicking on a tool allows the admin to edit an existing tool

Dashboard Interface

  • Once the user is logged in, they are greeted with the dashboard interface.
  • The navigation bar on the top contains the clickable links to the different features that GELMS offers.

Courses Page

  • This is the courses page which shows the courses that the student/teacher is registered to

Settings Page

  • This is the settings page
  • The user can change their password, turn on/off reader mode and turn on/off multifactor authentication here.(Note: Multifactor Authentication is currently not active and will be released in an upcoming version of GELMS)

Announcements Page

  • Students can view the announcements posted by the teacher for a particular course here
  • Teachers have the additional ability to add new announcements from this page.
  • Teachers can also edit or delete existing announcements from this page

Syllabus Page

  • Students can view the syllabus posted by the teacher for a particular course here
  • Teachers have the additional ability to add new syllabi items from this page.
  • Teachers can also edit or delete existing syllabi items from this page

People Page

  • This page shows the people registered to a particular course

Tools Page

  • This is the tools page
  • It shows a domain-specific software tool of the course that can be used by the student user
  • Teachers have the additional ability to lock or unlock the tool for that course.
